Ідея місії дуже проста. Тобі потрібно визначити, чи є дане ціле число паліндромом чи ні за основою B. Число є паліндромом, якщо воно читається однаково в обох напрямках, наприклад, 121 є паліндромом, а 122 – ні. Якщо ціле число є паліндромом за основою B, поверніть True, якщо ні, поверніть False.

Для вирішення місії використовуй тільки математичні операції над даним цілим числом.

Вхідні дані: Задане ціле число та основа B (ціле число).

Вихідні дані: Логічне (булеве) значення.
